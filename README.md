## 📁 Project Index

### ✅ Task 1 — Mobile Signup & Onboarding Flow
> **Folder:** [`Task-1_Mobile_Signup_Flow/`](./Task-1_Mobile_Signup_Flow/)

A complete mobile-first authentication UI Kit featuring 7 high-fidelity screens including Splash, Onboarding, Login, Signup, OTP Verification, and Welcome screens. Built with a full design system, component variants, and interactive prototype flows.

---

### ✅ Task 2 — Responsive Email Template Design
> **Folder:** [`Task-2_Email_Template/`](./Task-2_Email_Template/)

A premium SaaS email template system for FlowSphere — a modern productivity platform. Features a fully responsive layout for desktop (640px) and mobile (375px), interactive HTML prototype, 16-page Figma case study, and complete design token system.

---

### 🔄 Task 3 — Restaurant Menu UI Design
> **Folder:** [`Task-3_Restaurant_Menu/`](./Task-3_Restaurant_Menu/)

A modern, visually immersive restaurant menu app design. Features category navigation, food cards, detail screens, cart flow, and a clean ordering experience designed for mobile-first users.
